
# Título del proyecto

C# 12


## Detalles 
En este proyecto de consola se encuentra ejemplos del uso de las nuevas funcionalidades de c# 12.

Entre ellos encontramos los siguientes:
- Funciones locales
- Constructor principal
- parámetros lambda
- Parámetros predeterminados en lambda
